FAQ
Questions

Does it work with any git repository?
Yes. git-recap uses standard git commands under the hood. It works with any git repo, regardless of hosting platform (GitHub, GitLab, Bitbucket, self-hosted).
Does it send my code anywhere?
Absolutely not. git-recap runs entirely locally. It only reads your git history using git CLI commands. No data leaves your machine.
How does the license work?
After purchase, you receive a license key by email. Run git-recap activate YOUR-KEY to unlock Pro features. The key works offline — no license server required.
What counts as a "standup report"?
git-recap groups your commits by type (features, bug fixes, refactors, etc.) and shows stats (lines added/removed, files changed). It's exactly what you'd need to answer "what did you work on today?"
Can I use it without conventional commits?
Yes. If your commits don't follow conventional commit format, they show up under "Other". Conventional commit prefixes are auto-detected but not required.
